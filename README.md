# All of AI

인공지능 전 영역을 체계적으로 정리한 지식베이스입니다.

배포 URL: https://llouis0622.github.io/all-of-ai/

## 로컬 실행 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행 (http://localhost:5173/all-of-ai/)
npm run dev

# 정적 빌드
npm run build

# 빌드 결과 미리보기
npm run preview
```

## 기술 스택

- [VitePress](https://vitepress.dev/) — 정적 사이트 생성기
- [KaTeX](https://katex.org/) — 수식 렌더링
- [Mermaid](https://mermaid.js.org/) — 다이어그램
- GitHub Actions — 자동 배포
